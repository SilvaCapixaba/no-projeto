Oque são tags

Voltar

São elementos que definem conteúdos dentro do documento que estamos compondo que podem ser:

  1. Estrutura de pagina ou bloco (todo documento)

  2. Estrutura vizual de pagina (conteudo)

E como podemos entende-las

São apenas 4 tags que definem a estrututa do documento, que são elas:

  • DOCTYPE
  • HTML
  • HEAD
  • BODY
  • STYLE

  • DOCTYPE

É uma instrução para o navegador da web que diz em qual versão do html (quando tem html junto indica que é versão mais recente) a pagina foi escrita. E para gantir que pagina da web seja analizada e exibida da mesma maneira por nevegadores da web diferentes.

  • HTML

O elemento html também chamado de tags html, serve para informar ao navegador que tipo de estrututa esta sendo construida, podendo ser titulos, paragrafos, imagens, links, entre outros.

  • HEAD

Para a tradução portugues siguinifica 'Cabeça' dentro dela fica os dados conhecido com 'meta tags' que não aparecem no navegador, mas tambem trazem um conjuto de informações que especifica varios tipos de configurações para o navegador.(vamos especificar mais adiante) Dentro dela também colocamos tags de estilização, tags que traz estilo para nossa pagina.

  • BODY

Tem pelo seu siguinficado traduzido para português 'Corpo' dentro desta tag nós colocamos tudo oque e vizual para nossa pagina onde o usuário ve e interage tais como: texto, videos, imagens.

  • STYLE

Tag usada para dar estylo no conteudo desenvolvido, está tag e colocda no cabeçalho head.

Conclusão

Trazendo a nosso entendimento e clareza que temos uma parte da estrutura fixa que não altera, como as configurações principais de identificação dos buscadores onde sua pagina vai ser exibida em qualuer navegador. Uma outra parte e distribuida em colocar tags de exbição no corpo 'BODY' e tags de estylização dentro da tag 'HEAD'. Tags estas que vamos ver mais para frente ex: tags que é para mostrar uma imagem, video, musicas comando que vai montar sua pagina.
